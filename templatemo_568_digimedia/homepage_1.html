Objective

The majority of sectors have been automated in recent years, but the voting mechanism has not been altered in years. We suggest combining artificial intelligence (AI) with electronic voting machines (EVMs) to revolutionise the voting process.


Many expenditures associated with manual work at polling booths can be reduced with AI-powered EVMs. They also assist in the detection of fraud through the use of computer vision.

  • SQLite Database & Django Framework

    SQLite is used as a dataset to get processed with the data within an application. All eligible voting citizens are first registered into the database. Dashboards are created for administrators for maintaining records. Unregistered candidates are not eligible to vote.


    Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design is used for backend functioning.

    Optimized Records Data Info Data Analysis Ridiculously fast Reassuringly secure Exceedingly scalable

  • Person Detection Phase

    The MobileNet-SSD model is a Single-Shot multibox Detection (SSD) network. Pre-trained deep learning Model and can detect human even with partial exposure over video-cam.

    Real Time Detection 99% Precision Robust Integrity Agile Accuracy

    Count the number of person, if the number of people equals one, the following phase will begin otherwise candidate is not permitted to proceed further. It will be notified through pop-up and voice assistant.

  • Mask Detection Phase

    AI based Face Mask Detection Real time Detection Authentic Usability Ease of Maintainance Adaptability

    It prevents fraudulent entries. While voting, a person should remove their mask to allow the desired individual to vote. If a person is detected wearing a mask, a popup will appear and the voice assistant will alert the person.

  • Face Recognition Phase

    Face Recognition Library based on deep learning is used for face recognition. Image ROI extracts features from recognised faces using face recognition lib.

    Scalability Built-in privacy protection Accuracy Analytics Air-tight Matching Algorithm

    It will display the voter's name while recognizing the individual. If the voter's identify is found in the dataset, the following phase will be permitted. If the person is not found in the dataset, it will display an unknown identity message and offer popup and voice assistance to notify them. As a result, person will not be permitted to proceed to the following round.

  • Voting Phase

    Here the person can vote to their respected reprentatives of their choice!

    Singularility Authenticity Reassurance Auditability Privacy Transparency

    Voter will be informed via email displaying the name of the political party to whom they have voted.
